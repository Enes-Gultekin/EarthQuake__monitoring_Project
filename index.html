Earth-Quake spa application

This Single Page Application Project was inpired by recent Global Warming activities and Natural Disaster Events.


- On the left of the page, there are two buttons other than zoom buttons which represents attributes of layers.

- Search Button is simply used for searching any country.

- Table Button displays the table which has place,magnitude and date informations of every point every earthquake events .

- The Last button has two buttons inside: Significant EarthQuakes, Recent EarthQuake Events and Fault-Lines.

- On the right of the page, there is layer button which has two map layers and location button that shows current location.
